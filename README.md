# vue2.0-general

> General Components for Vue2.0.  [Demo](https://pengfu.github.io/vue2.0-general/)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
